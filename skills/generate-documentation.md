# generate-documentation

Skill para generar documentación estructurada de componentes o flujos directamente desde Figma, lista para exportar a Notion, Confluence o Storybook.

## Prerequisito
Carga figma-use antes de ejecutar este skill.

## Cuando usar este skill
- Cuando el usuario pide documentar un componente o sistema de diseño
- Cuando se necesita generar specs para el equipo de desarrollo
- Cuando se quiere crear una guía de uso de componentes
